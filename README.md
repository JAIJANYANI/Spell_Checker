# Software
* Tensorflow==1.1
* Numpy==1.11.3
* Pandas
* Sklearn
* Dash

Tested on Ubuntu 16.04 LTS amd64 xenial image built on 2017-09-19 40-core CPU 128 GB Ram

```
python Spell_Checker_Main.py
```
